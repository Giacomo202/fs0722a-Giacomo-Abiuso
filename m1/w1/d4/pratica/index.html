Come creare un template per i post

... di Mario Rossi - pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le (parti comuni colorato) da applicare a tutte le pagine

Le varie aree saranno quindi suddivise in:

  • Contenuti di intestazione del tag (header inquadrato
  • Contenuti centrali variabili del tag (main inquadrato)(corpo principale)
  • Contenuti di fine pagina comuni, da inserire nel tag (footer inquadrato)

Formattazione del codice
Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da (selezionare con i css colorato). Perciò i passi saranno:

  1. 1.Progettare su carta di markup finale, assegnando ad ogni specifico testo il tag appropriato
  2. 2.Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  3. 3.Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressocchè totale.
